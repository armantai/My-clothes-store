This is a starter full-stack clothes store (responsive web + backend).

Tech:
- Frontend: React (Vite), MUI, React Router
- Backend: Node.js, Express, MongoDB (Mongoose)
- Auth: JWT (email/password) + optional GitHub OAuth

To run locally:
- Backend: set environment variables (MONGO_URI, JWT_SECRET, GITHUB_CLIENT_ID, GITHUB_CLIENT_SECRET) then:
  cd backend
  npm install
  npm run dev

- Frontend:
  cd frontend
  npm install
  npm run dev
